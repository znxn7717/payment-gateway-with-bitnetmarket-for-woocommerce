درگاه پرداختی که کسب‌وکار شما را از وابستگی به بانک‌ها رها می‌کند. راه‌اندازی آنی، تراکنش‌های جهانی، آزادی مالی

== Description ==
درگاه پرداخت بیت‌نت‌مارکت امکان پذیرش پرداخت‌های رمزارزی را به سادگی و امنیت به فروشگاه شما اضافه می‌کند. با استفاده از این افزونه می‌توانید رمزارزهای مختلف را به عنوان روش پرداخت در فروشگاه خود فعال کرده و تجربه خریدی مدرن، سریع و بدون مرز برای مشتریان‌تان فراهم کنید

== Installation ==
1. پوشه `bitnetmarket-payment-gateway-for-woocommerce` را در `/wp-content/plugins/` آپلود کنید.
2. افزونه را فعال کنید.
3. در WooCommerce > Settings > Payments > Bitnetmarket شناسه فروشنده را وارد کنید.
